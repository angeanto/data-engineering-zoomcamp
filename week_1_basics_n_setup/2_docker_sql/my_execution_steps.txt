2. Create a postegres instance with docker and mounted my local file path with the db path 
 $ sudo docker run -it \
    -e POSTGRES_USER="root" \
    -e POSTGRES_PASSWORD="root" \
    -e POSTGRES_DB="ny_taxi" \
    -v $(pwd)/ny_taxi_postgres_data:/var/lib/postgresql/data \
    -p 5432:5432 \
    postgres:13

It worked. Above command starts the postgres.

6. Run pgadmin with docker

If port is binded: 

$ sudo netstat -pna | grep 8080
LISTEN      24116/utserver --> 24116 = PID
$ kill <pid>

or 
$ sudo ss -lptn 'sport = :5432'
$ kill <pid>

I had installed utserver to listen to this port. 
$ netstat -lnop --tcp | grep utserver
$ sudo rm -r /opt/utorrent-server-alpha-v3_3 /usr/bin/utserver

fixed the issue. 

$ sudo docker run -it \
  -e PGADMIN_DEFAULT_EMAIL="admin@admin.com" \
  -e PGADMIN_DEFAULT_PASSWORD="root" \
  -p 8080:80 \
  dpage/pgadmin4

7. We now have to create a pg-network in order to connect pgadmin container with postgres container. 
  $ sudo docker network create pg-network
  returns: 813d49f60590ef22233165608b3773bf0a9615c6814999bc971407652da9600e
  and we added: 

  --network=pg-network \
  --name pg-database \

finally: 

$ sudo docker run -it \
         -e POSTGRES_USER="root" \
         -e POSTGRES_PASSWORD="root" \
         -e POSTGRES_DB="ny_taxi" \
         -v $(pwd)/ny_taxi_postgres_data:/var/lib/postgresql/data \
         -p 5432:5432 \
         --network=pg-network \
         --name pg-database \
         postgres:13

Same for pgadmin container: 

$ sudo docker run -it \
  -e PGADMIN_DEFAULT_EMAIL="admin@admin.com" \
  -e PGADMIN_DEFAULT_PASSWORD="root" \

====== My Queries ============
SELECT COUNT(*) 
FROM public.yellow_taxi_data

SELECT
MIN(tpep_pickup_datetime) AS min_date,
MAX(tpep_pickup_datetime) AS max_date
FROM public.yellow_taxi_data

SELECT
DISTINCT(DATE_TRUNC('day',tpep_pickup_datetime))
FROM public.yellow_taxi_data
WHERE date_part('month', tpep_pickup_datetime) = 1
AND date_part('day', tpep_pickup_datetime) = 15
AND date_part('year', tpep_pickup_datetime) = 2021

SELECT
COUNT(*)
FROM public.yellow_taxi_data
WHERE date_part('month', tpep_pickup_datetime) = 1
AND date_part('day', tpep_pickup_datetime) = 15
AND date_part('year', tpep_pickup_datetime) = 2021
